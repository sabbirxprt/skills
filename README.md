# wp-plugin-review

Audits WordPress plugins for WordPress.org compliance and security. Use when reviewing a plugin for submission, preparing for plugin review, or auditing plugin code for guidelines and security.

**Install in Cursor:** `/learn @sabbirxprt/wp-plugin-review`
